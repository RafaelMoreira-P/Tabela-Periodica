# Tabela Periódica

Este projeto consiste na construção de uma Tabela Periódica utilizando tecnologias básicas de desenvolvimento web.
A estrutura dos elementos é organizada de forma visual para representar a disposição real dos elementos químicos.

# Objetivo
O projeto foi desenvolvido com o objetivo de praticar conceitos de estruturação de páginas, layout com CSS e organização visual de dados.

Além disso, o projeto também ajuda a compreender como dados estruturados podem ser organizados visualmente em uma interface web.

Este projeto foi desenvolvido para fins educacionais e de aprendizado em desenvolvimento web.
